# FRONTEND-PROJECT-LVL2
## Gendiff 
Gendiff it is a cli utility that can help you to generate differens betwen two json, yaml or ini files. 
### Requirements
* Mac/Linux
* Node & npm
### Setup

```
$ make install
```
### Run tests

```
$ make test
```
